4 Robust

Content must be robust enough to be reliably parsed by a variety of user agents

4.1 Compatible

4.1.1 Parsing: Code must be good. Validate site using HTML validators.

4.1.2 Name, Role, Value: Interface components should be able to programmatically be determined as to what they do (role) and what their context is (name), as well as possible state or value.
