3 Understandable

3.1 Readable

3.1.1 Language of Page: Set the lang attribute on the html tag

3.1.2 Language of Parts: Set the lang attribute on individual tags if language is different than page.

3.2 Predictable

3.2.1 On Focus: Don't do weird things (e.g. open a new window) when an element recieves focus

3.2.2 On Input: Don't change context unless user is informed of intent. For example hiding form elements after input on a select form element.

3.2.3 Consistant Navigation: Don't change placement or order of navigation

3.2.4 Consistent Identification: Items on a page are labelled in a consistant manner. For example - a next page button would read "Go to Page [X]" on every page.

3.3 Input Assistance

3.2.1 Error Identification: If an error is detected, identify error and describe error in text

3.2.2 Labels or Instructions: Label fields and provide instructions on how to fill out form

3.2.3 Error Suggestion: provide suggestions on how to correctly fill out form.

3.2.4 Error Prevention (Legal, Financial, Data): If a user is submitting contract, legal data, modifying records in a DB - add sufficient controls to confirm data is correct.
