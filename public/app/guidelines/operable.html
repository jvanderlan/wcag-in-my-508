2 Operable

2.1 Keyboard Accessible

2.1.1 Keyboard All page functionality must work via keyboard.

2.1.2 No Keyboard Trap All of the page must be navigable via keyboard.

2.2 Enough Time

2.2.1 Timing Adjustable Must be able to alter timing, stop timer, extend deadline unless it's real-time (auction), essential to function (test), or longer than 20hrs.

2.2.2 Pause, Stop, Hide Moving, scrolling, blinking or auto-updating elements that start automatically have to have controls to pause, stop or hide it.

2.3 Seizures

2.3.1 Three Flashes or Below Threshold: Nothing should flash more than 3 times per second or the flash is below the general flash and red flash thresholds.

2.3.2 Three Flashes: Nothing should flash more than 3 times per second

2.4 Navigable

2.4.1 Bypass Blocks: Skip links to bypass blocks of repeated content

2.4.2 Page Titled: Page has title tag

2.4.3 Focus Order: Page can be navigated in a sequential order that preserves meaning and operability.

2.4.4 Link Purpose (In Context): Purpose of a link can be derived from link text or the link's context.

2.4.5 Multiple Ways:Provide several different means to navigate site, main navs, search forms, sitemaps.

2.4.6 Headings and Labels: Describe topic or purpose

2.4.7 Focus Visible: Any keyboard operable element should have visible focus.
