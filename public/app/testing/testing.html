Testing

Great list of tools 18f Accessibility Guide

For general testing, install WAVE toolbar.

1 Perceivable

Turn off CSS to make sure information flow makes sense and correct tags are being used to present information (h tags, list tags, etc.)

Use a screen reader, such as NVDA, to make sure all content is presented to visually impaired users.

Use the Color Contrast Analyzer Chrome Plugin to test 4.5:1 ratios

WebAIM's Color Contrast Checker is also handy.

2 Operable

Use keyboard to tab through the entire page - top to bottom. Make sure order is logical, focused element looks the same as if it were mouseover.

Use HTML5 Outliner to make sure all regions have a header.

3 Understandable

Use WAVE toolbar to check for errors (e.g. missing language).

Tab through and use form elements. Make sure nothing happens that isn't described to user.

4 Robust

Validate your HTML document with Web Developer Toolbar.

Use WAVE toolbar to audit if your document has adequate ARIA attributes to convey roles and names. Look for missing form labels.
